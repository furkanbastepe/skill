Mastering the 'for' Loop

In this challenge, you will write a loop that iterates through a provided array of numbers. Your goal is to sum only the even numbers.

checklist Task Requirements

  • Create a variable named sum initialized to 0.
  • Use a for loop to iterate through the numbers array.
  • Check if the current number is even using the modulo operator %.
lightbulb Complexity Hint: Big O
expand_more

A single loop that touches every element once is considered Linear Time or O(n).

Be careful with nested loops! If you put a loop inside another loop, your complexity becomes O(n²), which is much slower for large datasets.

// Challenge: Sum all even numbers in the array
let numbers = [12, 45, 23, 8, 4, 101, 56];
let sum = 0;

// TODO: Write your loop below
for (let i = 0; i < numbers.length; i++) {
    let currentNum = numbers[i];
    
    // Check if even
    if (currentNum % 2 === 0) {
        sum += currentNum;
        console.log("Added even number: " + currentNum);
    }
}

console.log("Final Sum: " + sum);
